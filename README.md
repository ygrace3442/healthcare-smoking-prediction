# 헬스케어 데이터 기반 흡연 여부 예측 모델

본 레포지토리는 건강검진 데이터를 활용해 **흡연 여부를 예측하는 다중 클래스 분류 모델**을 구축한 프로젝트입니다.  
단순한 모델 구현을 넘어, 실제 헬스케어 데이터를 대상으로 한 **엔드 투 엔드 머신러닝 파이프라인**을 설계하고 검증하는 것을 목표로 합니다.

- 헬스케어 데이터 EDA 및 시각화
- 결측치·이상치 처리 등 데이터 전처리
- 도메인 지식을 반영한 피처 엔지니어링
- 전통적인 머신러닝 모델 벤치마킹
- **AutoGluon Tabular**를 활용한 AutoML 및 스태킹 앙상블

---

## 1. 프로젝트 개요

- **문제 유형**: 다중 클래스 분류 (흡연 여부 예측)
- **도메인**: 건강검진 / 예방의학 데이터 분석
- **목표**
  - 일반화 성능이 우수한 흡연 여부 예측 모델 구축
  - 데이터 전처리·피처 엔지니어링·AutoML의 효과를 정량적으로 비교
  - 헬스케어 데이터에 특화된 분석·모델링 흐름 정리

### 1.1 사용 데이터(개요)

본 레포지토리에는 **원본 데이터가 포함되어 있지 않습니다.**  
데이터셋은 건강검진 데이터를 기반으로 하며, 대략 다음과 같은 변수를 포함합니다.

- **인구학적 정보**: 나이, 성별
- **신체 계측**: 키, 몸무게, BMI
- **혈압**
- **혈액 검사**  
  - 공복 혈당, 중성지방, 총 콜레스테롤  
  - HDL, LDL, 혈청 크레아티닌, 헤모글로빈 등
- **간 효소** 및 기타 생화학 지표

원본 데이터는 대회/공개 플랫폼 등의 출처에서 직접 다운로드해야 하며,  
사용 시 해당 출처의 **라이선스 및 이용 약관을 반드시 준수해야 합니다.**
