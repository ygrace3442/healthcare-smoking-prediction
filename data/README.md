# Data Guide

The original health-check dataset used in this project is **not included** in this repository
due to license / competition policy.

Please download the dataset from the original source (e.g., Dacon healthcare smoking prediction competition)
and place the files as follows:

- `data/train.csv`
- `data/test.csv`

Then, open the notebook in `notebooks/smoking_prediction_autogluon.ipynb`
and adjust the file paths if necessary.
